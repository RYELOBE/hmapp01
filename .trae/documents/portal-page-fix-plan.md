## 路由修复说明

### 当前路由配置问题
```javascript
// 当前配置 - 多个路由使用同一组件
{ path: "home", component: PortalHome }
{ path: "buyer/items", component: PortalHome }  // Bug：应该使用独立组件
```

### 修复后配置
```javascript
// 修复后 - 使用不同组件
{ path: "home", component: PortalHome }  // 首页 - 只展示热门
{ path: "buyer/items", component: ItemsPage }  // 全部商品 - 完整列表
```

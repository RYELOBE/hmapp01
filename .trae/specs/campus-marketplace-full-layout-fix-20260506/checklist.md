# 校园二手交易平台 - 完整页面框架修复 - 验证清单

## 布局结构验证

- [ ] 顶部导航栏正确显示
  - **验证方法**：访问 `/portal/home`，查看页面顶部
  - **预期结果**：
    - 导航栏高度64px，白底/浅蓝底
    - 左侧显示Logo "校园二手平台" + 主导航菜单
    - 右侧显示搜索/通知图标 + 用户信息区域
    - 导航栏固定在顶部（sticky定位）
  - **优先级**：P0

- [ ] 页面层级关系正确
  - **验证方法**：检查页面DOM结构和视觉层次
  - **预期结果**：
    - Header在最顶层（z-index高）
    - Hero区域在Header下方，全宽显示
    - 内容区域正常排列，无遮挡问题
  - **优先级**：P0

## 导航功能验证

- [ ] 所有导航链接可点击
  - **验证方法**：依次点击每个导航项（首页、全部商品、卖家精选等）
  - **预期结果**：
    - 点击后有hover效果反馈
    - URL正确更新为对应路由
    - 页面跳转无延迟或卡顿
  - **优先级**：P0

- [ ] 当前页面导航项高亮
  - **验证方法**：在不同页面间切换，观察导航栏状态
  - **预期结果**：
    - 当前路由对应的导航项有明显的激活样式（颜色/背景/下划线）
    - 切换页面后，高亮状态正确转移
  - **优先级**：P1

- [ ] 用户登录状态显示正确
  - **验证方法**：分别测试未登录和已登录状态
  - **预期结果**：
    - 未登录：右侧显示"登录 / 注册"按钮，点击跳转登录页
    - 已登录：显示用户头像 + 用户名 + 下拉菜单（个人中心、退出）
  - **优先级**：P0

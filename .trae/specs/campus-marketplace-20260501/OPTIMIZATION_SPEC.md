### 4.4 布局规范

**页面容器**
- 内边距: 20px
- 卡片间距: 16px
- 卡片圆角: 8px

**栅格系统**
- 列数: 24
- 间距: 16px
- 断点: 576px / 768px / 992px / 1200px / 1400px

**响应式断点**
- xs: < 576px (手机)
- sm: 576px - 768px (平板)
- md: 768px - 992px (小屏电脑)
- lg: 992px - 1200px (电脑)
- xl: > 1200px (大屏)

### 4.5 动画规范

**过渡时长**
- 快速: 0.15s
- 正常: 0.3s
- 慢速: 0.5s

**缓动函数**
- 默认: ease
- 进入: ease-out
- 退出: ease-in
- 弹性: cubic-bezier(0.68, -0.55, 0.27, 1.55)

## 五、技术实现规范

### 5.1 前端规范

**Vue组件规范**
- 使用组合式API (<script setup>)
- 组件名称使用 PascalCase
- Props使用 defineProps
- 事件使用 defineEmits
- 样式使用 scoped

**文件命名规范**
- Vue组件: PascalCase (如: UserList.vue)
- JS文件: camelCase (如: userList.js)
- CSS文件: kebab-case (如: user-list.css)
- 常量文件: UPPER_SNAKE_CASE

**目录结构**
```
src/
├── assets/          # 静态资源
├── components/       # 公共组件
├── composables/      # 组合式函数
├── layouts/         # 布局组件
├── router/          # 路由配置
├── services/        # API服务
├── stores/          # 状态管理
├── utils/           # 工具函数
└── views/           # 页面组件
```

### 5.2 后端规范

**包结构**
```
com.campus.marketplace/
├── config/          # 配置类
├── controller/      # 控制器
├── service/         # 业务服务
├── repository/      # 数据访问
├── entity/          # 实体类
├── dto/             # 数据传输对象
├── vo/              # 视图对象
├── exception/       # 异常类
└── util/           # 工具类
```

**命名规范**
- 类名: PascalCase
- 方法名: camelCase
- 常量: UPPER_SNAKE_CASE
- 数据库表/字段: snake_case

### 5.3 API规范

**请求规范**
- Content-Type: application/json
- 认证: Bearer Token

**响应规范**
```json
{
  "code": 200,
  "message": "操作成功",
  "data": {}
}
```

**分页响应**
```json
{
  "code": 200,
  "data": {
    "list": [],
    "total": 100,
    "page": 1,
    "pageSize": 10
  }
}
```

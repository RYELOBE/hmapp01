# 校园二手交易平台 - 产品需求文档（PRD）

## Overview
- **Summary**: 校园二手交易平台是面向高校学生的综合性二手物品交易系统，提供商品发布、浏览、交易、评价等核心功能，并包含完善的审批流程确保内容质量。
- **Purpose**: 解决校园内二手物品交易信息不对称、交易安全缺乏保障等问题，打造一个便捷、安全、可靠的校园二手交易生态圈。
- **Target Users**: 在校大学生（买家/卖家）、平台运营人员

## Goals
- 建立完整的商品发布与审核机制
- 实现评价功能与评价审批流程
- 提供收藏功能增强用户粘性
- 构建统一的审批流管理体系
- 打造消息中心实现高效信息推送
- 创建校园圈子促进社区互动
- 商品展示采用瀑布流布局
- 完善用户角色体系（买家、卖家、运营）

## Non-Goals (Out of Scope)
- 第三方支付集成（仅提供交易撮合）
- 物流配送服务
- 跨校交易功能
- 移动端App开发（优先Web端）

## Background & Context
- 参考设计：https://datainfra.szdex.com/#/platformportal/portalcj-web/home
- 现有页面需升级优化，采用更现代化设计风格
- 需要新增评价、收藏、审批流等功能

## Functional Requirements
- **FR-1**: 用户可注册登录，支持买家/卖家角色切换
- **FR-2**: 商品发布需填写详细信息，支持富文本编辑
- **FR-3**: 商品发布后需经过运营审核才能展示
- **FR-4**: 商品列表采用瀑布流布局展示
- **FR-5**: 用户可对已购买商品进行评价
- **FR-6**: 评价需经过审批才能展示
- **FR-7**: 用户可收藏商品，管理收藏列表
- **FR-8**: 运营人员可进行商品审核操作
- **FR-9**: 运营人员可进行评价审核操作
- **FR-10**: 消息中心支持系统消息发布与推送
- **FR-11**: 校园圈子支持帖子发布与互动
- **FR-12**: 帖子与评论需经过审批才能展示
- **FR-13**: 关于我们页面展示平台介绍信息

## Non-Functional Requirements
- **NFR-1**: 页面响应时间 ≤ 2s，接口响应时间 ≤ 500ms
- **NFR-2**: 支持1000+并发用户
- **NFR-3**: 用户密码加密存储，接口需Token鉴权
- **NFR-4**: 支持主流浏览器：Chrome、Firefox、Safari、Edge
- **NFR-5**: 响应式设计，支持PC端和移动端

## Constraints
- **Technical**: Vue.js 3 + Spring Boot 3 技术栈
- **Business**: 面向高校学生群体，需符合校园文化氛围
- **Dependencies**: 需集成富文本编辑器、图片存储服务

## Assumptions
- 用户已注册并完成身份认证
- 运营人员具备审核权限
- 系统部署在稳定的服务器环境

## Acceptance Criteria

### AC-1: 商品发布与审核流程
- **Given**: 用户已登录且为卖家身份
- **When**: 用户填写商品信息并提交审核
- **Then**: 商品进入待审核状态，运营人员可见待审核列表
- **Verification**: `programmatic`
- **Notes**: 商品信息包含名称、描述（富文本）、价格、图片、分类

### AC-2: 商品审核操作
- **Given**: 运营人员登录系统
- **When**: 运营人员查看待审核商品并选择通过/驳回
- **Then**: 商品状态更新，通过则展示到商品列表，驳回则通知用户
- **Verification**: `programmatic`

### AC-3: 商品瀑布流展示
- **Given**: 用户访问商品列表页
- **When**: 页面加载完成
- **Then**: 商品以瀑布流布局展示，支持筛选和搜索
- **Verification**: `human-judgment`

### AC-4: 评价发布与审批
- **Given**: 用户已购买商品
- **When**: 用户提交评价内容和评分
- **Then**: 评价进入待审核状态，运营人员可审核
- **Verification**: `programmatic`

### AC-5: 收藏功能
- **Given**: 用户已登录
- **When**: 用户点击商品收藏按钮
- **Then**: 商品加入用户收藏列表
- **Verification**: `programmatic`

### AC-6: 校园圈子发帖
- **Given**: 用户已登录
- **When**: 用户发布圈子帖子（富文本+图片）
- **Then**: 帖子进入待审核状态，审核通过后展示
- **Verification**: `programmatic`

### AC-7: 消息中心
- **Given**: 运营人员登录系统
- **When**: 运营人员发布系统消息
- **Then**: 消息进入待审核，审核通过后推送给用户
- **Verification**: `programmatic`

### AC-8: 关于我们页面
- **Given**: 用户访问关于我们页面
- **When**: 页面加载完成
- **Then**: 展示平台介绍、团队信息、联系方式等
- **Verification**: `human-judgment`

### AC-9: 用户角色区分
- **Given**: 不同角色用户登录
- **When**: 访问系统功能
- **Then**: 买家仅能浏览购买，卖家可发布商品，运营可审核管理
- **Verification**: `programmatic`
